# GAS API 連携仕様 (在庫管理アプリ)

Google Apps Script (GAS) を使用して、Google スプレッドシートとデータの同期・ログ記録を行うための仕様です。

## 概要
- **目的**: localStorage データのバックアップ、クラウド同期、在庫操作の履歴記録。
- **エンドポイント**: `localStorage` の `inventory_app_gas_url` に保存された Web アプリ URL。

## API リクエスト形式
GAS へのリクエストは `POST` メソッドで行い、JSON データを送信します。

### 1. データ同期 (Sync)
全商品データやカテゴリデータをスプレッドシートに保存、または読み込みます。

### 2. 在庫ログ記録 (Log)
在庫の増減が発生した際に、その履歴をスプレッドシートの「ログ」シートに追記します。

## 実装上の注意
- **同期のタイミング**: 設定画面からの手動同期、またはアプリ起動時に自動同期を試行します。
- **オフライン対応**: オフライン時は localStorage にキューを貯め、次回オンライン時にまとめて送信する仕組みを検討中です。
- **GAS側の設定**: GAS のプロジェクトで「ウェブアプリとしてデプロイ」し、アクセス権限を「全員（匿名を含む）」に設定する必要があります。
